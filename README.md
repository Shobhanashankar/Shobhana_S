### 🛡 What I work on

- 📞 **EchoShield** – VoIP Security & Intelligence Dashboard  
  - Dashboard for SIP/RTP traffic, speech recognition, speaker diarization, spoof detection, and spam IP tracking for real-time call risk scoring.  

- 🧬 **Malware Classification via Assembly Instructions** – Analyst Console  
  - Web console to upload samples, run opcode/inst–based ML classification, and view SHAP-driven explanations for malware family decisions.  

- 🌐 **Real-time DNS Anomaly Detection** – Network Monitoring Dashboard  
  - Live DNS monitoring dashboard using Scapy sniffing, entropy/structure features, and Isolation Forest to surface suspected DGA/tunneling activity.  

- 🔍 **CALLGraph OSINT – Incident Investigation Console**  
  - Investigation dashboard that correlates SIP logs, PCAPs, and IP/domain intel APIs into a single triage view for analysts.  

- 📚 **EduBotAI+** – Study Assistant Dashboard on Azure

- 🧠 **SecondBrain** – Developer Knowledge Base Dashboard
